Para a aplicação funcionar, siga os seguintes passos 
	
	1) Importar o projeto na IDE Netbeans
	2) Clicar com o botão direto no Projeto e Executar
	3) Inserir o ID de um usuário da lista de teste(abaixo)
	4) Inserir o ID de um filme que você deseja saber o quanto o usuário vai gostar. Valores possíveis: 1 a 20
	5) Observação. O calculo da nota estimada pode demorar, devido as pesquisas no banco.
	6) [Opcional] para gráficos e print das regras, clicar com o botão direto na classe FuzzyMovies e clicar em Executar 	   Arquivo
